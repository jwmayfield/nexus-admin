nexus-admin
--------------

Provides the module for the Django admin in `Nexus <https://github.com/dcramer/nexus>`_.

Config
======

Add nexus_admin to your ``INSTALLED_APPS``::

	INSTALLED_APPS = (
	    'nexus',
	    'nexus_admin',
	)

Usage
=====

The admin module is automatically integrated into Nexus.
